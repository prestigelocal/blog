base64-url

Today I wanted to explore whether I could add a "Share" feature on a markdown editor I'm building called apolloskine.

I knew that base64 encoding is an acceptable solution. But does this string need to be encoded to be parsed in a url? Turns out it does which is where the npm package base64-url comes along. 

      import base64url from 'base64-url';

      const decodedFromText = base64url.encode(content);
      const parsedFromUrl = base64url.decode(content);

I'm assuming there are some limitations with this approach, like potentially a limit on the url length? I'm sure its fine for my simple note taking app.
